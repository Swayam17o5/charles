FTP (File Transfer Protocol) is an application-layer protocol that allows files to be transferred between a client and a server over a TCP/IP network.
It uses port 21 for control and port 20 for data transfer.

vsftpd (Very Secure FTP Daemon) is a secure and fast FTP server for Linux. It supports both active and passive modes and provides user authentication, file upload/download, and directory isolation (chroot jail).

Key Configuration Parameters

local_enable=YES – allows local users to log in

write_enable=YES – enables upload/delete operations

chroot_local_user=YES – locks user to home directory

allow_writeable_chroot=YES – permits write access inside home
